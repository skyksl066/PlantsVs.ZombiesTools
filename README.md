# PlantsVs.ZombiesTools
使用[Cheat Engine](https://www.cheatengine.org/)定位遊戲參數記憶體位置並使用該位置開發程式修改遊戲參數

# Environment
- WinForm
- .NET Framework 4.7.2
- Windows 10 x64
- [Plants vs Zombies](https://www.ea.com/games/plants-vs-zombies/plants-vs-zombies)
- Visual Studio 2022

# Installation
Just clone and build project.
